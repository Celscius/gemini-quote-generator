# Quote-generator

simple quote generator made with gemini api

Quick usage:
create config.js file in root directory with following code
```js
// config.js
export default {
	API_KEY: "your-gemini-api-here",
	SUBJECT: "How's Your Day?"
}
```
if you have node js installed then yoo can use ```http-server``` and open ```http://127.0.0.1:8080/``` on your browser.
